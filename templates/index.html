servec

A lightweight HTTP server written in C

Simple, fast, and straightforward. Perfect for development and serving static files with minimal overhead.

Features Architecture

Core Features

Thread Pool

Multi-threaded request handling for concurrent connections. Configurable worker count.

Security

Directory traversal protection, extension whitelisting, and blocked sensitive paths (.git, .env, src/).

Configuration

INI-style config files with custom error pages, MIME type detection, and port configuration.
